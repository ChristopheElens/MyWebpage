Webdesign

Tijdens de lessen Webtechnologie leren wij om een website te schrijven: met behulp van Bootstrap, CSS en HTML-code, bouwen wij stapsgewijs onze website op. Deze website is onderdeel van onze cursus.

De kunst van Webdesign

Het opbouwen van een website is een langdurend project in verschillende fasen. Van het maken van de eerste schets, tot het uiteindelijk publiceren op het internet is een lang proces die uit verschillende fasen bestaat.
Vooral het permanente controleren en evalueren tijdens elke stap, maakt dat er veel tijd opgaat tussen idee en werkelijkheid. Hieronder schets ik even kort enkele van deze stappen die gemaakt worden tijdens het opbouwen van een website.

Wireframes

Een Wireframe is een soort 'schets' van hoe je een website zou willen opbouwen. Een soort 'getekende' versie van hoe jij jouw website wilt laten uitzien. Het is altijd belangrijk om een eindresultaat voor ogen te houden. Zeker bij een tijdsintensieve opdracht.

Hier zie je een eerste schets van hoe ik de website wilde opbouwen. Doorheen het proces, veranderd de layout o.i.v. nieuwe ideeën en inzichten. De eerste schets lijkt dan ook helemaal niet meer op het resultaat dat je nu aan het lezen bent.

HTML

HTML is de programeertaal waarmee websites geschreven worden. De code is zodanig eenvoudig dat die zelfs kan geschreven worden in simpele tekst-editors zoals Kladblok. Voor deze website gebuiken we Microsoft's Visual Studio © : een programmeeromgeving die handig gebruikt kan worden om in meerdere programmeertalen te schrijven, zoals ook Python ©

Zoals je op de afbeelding kunt zien, bestaat zo'n code uit verschillende elementen, die elk een kort omschrijven hoe de webbrowser elk element moet interpreteren en correct weergeven. Eventuele eigenschappen van elk element wordt omschreven in een 'class'. De opmaakregels hiervoor worden geschreven in nog een ander document: een CSS-bestand.

Online

Als de website uiteindelijk afgewerkt raakt, moet deze natuurlijk nog online gezet worden. Hiervoor zijn er verschillende platformen waar je dit op kan doen.
