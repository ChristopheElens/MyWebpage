Scripting

In de lessen Scripting: intro, moeten wij de beginselen van het programmeren inoefenen door kleine programma's te schrijven. Dit is een goede oefening om later kleine scriptbestandjes te schrijven die allerlei taken automatisch uitvoert op de server.

Wij programmeren in onze lessen voornamelijk in Python ©. Een programeertaal die steeds vaker gebruikt wordt in de programeerwereld. Met behulp van Microsoft's Visual Studio ©, een programeer-editor, bouwen wij stukjes code op om allerlei opdrachten uit te laten voeren op computersystemen.
